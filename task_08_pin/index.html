Task 8 — 5-Digit PIN Codes Simulation

This simulation generates random 5-digit PIN codes. Each digit is chosen independently from 0–9 (leading zeros allowed, e.g. 00123). Each code is checked to see whether all digits are different or if some digits repeat. This helps visualize the difference between sequences with repetition and without repetition. With more samples, the observed percentages approach the theoretical probabilities.

📐 Theory

  • Total codes: 10⁵ = 100,000 (each digit 0–9, repetition allowed)
  • All distinct: 10×9×8×7×6 = 30,240 → 30.24%
  • With repetition: 100,000 − 30,240 = 69,760 → 69.76%

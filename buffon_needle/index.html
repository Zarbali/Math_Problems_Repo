Task 5

Buffon's Needle
Experiment

Probability · Continuous Sample Spaces · Monte Carlo

00 Experiment

A needle of length L is thrown randomly onto a floor with equally spaced parallel lines. The distance between neighboring lines is d.

Right away I noticed this problem feels different from the previous ones. In earlier tasks we always had discrete outcomes — coin tosses, die rolls, drawing cards. Here the needle can land in infinitely many positions and angles.

That means the sample space will be continuous — we cannot list all possible outcomes individually.
01 Sample Space Ω

The sample space Ω contains all possible outcomes of a single throw. I asked myself: what information is actually needed to fully describe where the needle landed? Two quantities are enough.

02 The Two Parameters
ParameterMeaning
xDistance from needle's center to nearest line
θAngle between needle and the parallel lines

Once we know both the position of the center and the orientation, the entire configuration is determined.

03a Position — x

Because the lines repeat every distance d, I only need to track position within one strip. By symmetry I restrict to half the strip:

x ∈ [0, d/2]

x = 0 → center on a line  |  x = d/2 → center halfway between lines

03b Orientation — θ

Rotating the needle by 180° gives the same physical configuration — so I only need half a full rotation:

θ ∈ [0, π/2]

θ = 0 → parallel to lines  |  θ = π/2 → perpendicular

04 Sample Space as a Set

Each elementary outcome is a pair (x, θ). The full sample space:

Ω = { (x, θ) : x ∈ [0, d/2], θ ∈ [0, π/2] }
Ω = [0, d/2] × [0, π/2]

Geometrically this is a rectangle — every point inside corresponds to one possible outcome.

05 Why Continuous
  • Coin toss → 2 outcomes
  • Die → 6 outcomes
  • Cards → finite outcomes
  • Weather → finite outcomes

Here both x and θ can take infinitely many values — 0.1, 0.101, 0.10123… no complete list is possible.

∴ Sample space is continuous
A single exact outcome (x, θ) has probability zero. Probabilities are calculated over regions using probability density and integration.
